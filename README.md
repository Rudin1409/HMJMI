## 📖 About The Project

HMJMI Website is a modern web platform developed to support the digital transformation of the Himpunan Mahasiswa Jurusan Manajemen Informatika.

The platform provides centralized access to organizational information, announcements, events, documentation, and communication channels for students and members.

This project is actively maintained and continuously improved to provide a better experience for users while serving as a practical example of modern web development practices.

## 🚀 Installation

Clone repository:

```bash
git clone https://github.com/Rudin1409/HMJMI.git
```

Move to project folder:

```bash
cd HMJMI
```

Install dependencies:

```bash
npm install
```

Run development server:

```bash
npm run dev
```

Open browser:

```bash
http://localhost:3000
```

---

## 🔧 Environment Variables

Create a `.env.local` file and configure:

```env
NEXT_PUBLIC_FIREBASE_API_KEY=
NEXT_PUBLIC_FIREBASE_AUTH_DOMAIN=
NEXT_PUBLIC_FIREBASE_PROJECT_ID=
NEXT_PUBLIC_FIREBASE_STORAGE_BUCKET=
NEXT_PUBLIC_FIREBASE_MESSAGING_SENDER_ID=
NEXT_PUBLIC_FIREBASE_APP_ID=
```
